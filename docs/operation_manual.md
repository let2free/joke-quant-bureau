## 🔧 数据流架构

```
通达信TDX → etf_data.py → ETF_CACHE(120s) → 前端API
                                            ↓
九章流程 → artifacts/日期/ → fusion_report.json → /api/predictions/*
                                            ↓
数据累积 → accuracy_history.json → /api/accuracy/*
          ↓
         SQLite(database.py) → /api/factors/* + /api/stats
```

## ⚠️ 故障排查

| 症状 | 原因 | 解决 |
|------|------|------|
| 页面空白 | 服务器未启动 | 说 **"启动看板"** |
| ETF数据显示"--" | TDX非交易时段 | 等待开盘(9:30) |
| 回测无数据 | history.jsonl空 | 积累10天以上数据 |
| 图表不显示 | Chart.js CDN失败 | 刷新页面 |
| 雷达图重叠 | 数据接近 | 正常(三只ETF因子接近) |
