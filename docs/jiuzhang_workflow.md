## 预测流程（每日09:30）

### Phase 0: 系统性因子预检（必须）
```
输入：通达信实时数据
输出：市场状态判断（宽松/中性/紧张）
工具：tdx_quotes, tdx_screener
```

1. 获取A股大盘指数（上证/深证/创业板/科创50）
2. 检测L0流动性状态
   - 央行OMO/MLF操作
   - DR007利率
   - 北向资金流向
3. 检测L4b微观结构冲击
   - 指数调仓事件
   - 被动资金流入
4. 输出市场状态判断

### Phase 1: 并行侦察（3路并发）
```
Agent：大A + 老美 + 军机处
输入：Phase 0输出 + 各自领域数据
输出：各领域ETF候选清单
```

**大A（A股市场哨兵）**
- 分析A股动态、板块轮动、政策信号
- 输出：A股ETF候选清单（含L1-L5评估）

**老美（美股市场哨兵）**
- 分析美股动态、中概股、美联储信号
- 输出：美股/海外ETF候选清单

**军机处（全球宏观情报）**
- 分析地缘政治、大宗商品、汇率
- 输出：宏观环境评估 + 风险提示

### Phase 2: 深度分析（2路并发）
```
Agent：量化1哥 + 数据工程师
输入：Phase 1输出 + 通达信数据
输出：因子打分 + 数据验证
```

**量化1哥（因子打分）**
- 计算6大类因子得分
  - 动量因子（35%）
  - 资金流因子（25%）
  - 均值回归因子（15%）
  - 波动率因子（8%）
  - 微观结构因子（7%）
  - 市场状态因子（10%）
- 输出：ETF因子打分排名

**数据工程师（数据验证）**
- 验证通达信数据准确性
- 检查数据完整性
- 输出：数据质量报告

### Phase 3: 交叉验证（1路）
```
Agent：因子研究员
输入：Phase 2输出 + 历史数据
输出：IC计算 + 权重迭代建议
```

- 计算因子IC值（信息系数）
- 分析因子有效性
- 提出权重调整建议
- 输出：因子有效性报告

### Phase 4: 风控审查（1路）
```
Agent：包青天
输入：Phase 2输出 + 风控规则
输出：风控审查报告
```

- 审查L7买入纪律
- 评估尾部风险
- 检查止损设置
- 输出：风控建议

### Phase 5: 双轨融合 + 最终输出
```
Agent：指挥官
输入：所有Phase输出
输出：最终预测报告
```

**Track A: AI定性分析**
- 汇总大A + 老美 + 军机处的输出
- 结合L0/L4b调整

**Track B: 数学因子打分**
- 使用量化1哥的因子打分
- 应用权重迭代建议

**双轨融合**
- Track A × 0.4 + Track B × 0.6 = 融合分
- 一致性确认：双轨一致 → 高置信度
- 分歧点标记：双轨分歧 → 中/低置信度

**最终输出**
- Top 6 ETF推荐（含校准后预测幅度）
- 校准系数：原始预测 × 0.6
- 保存到 `artifacts/YYYY-MM-DD/fusion_report.json`

## 复盘流程（每日15:30）

### Phase 0: 数据采集
```
输入：今早预测结果 + 通达信实际数据
输出：预测 vs 实际对比数据
```

1. 读取预测结果 `artifacts/YYYY-MM-DD/fusion_report.json`
2. 获取今日实际收盘数据
3. 计算预测准确率

### Phase 1: 并行分析（3路并发）
```
Agent：大A + 老美 + 军机处
输入：今日实际表现 + 预测差异
输出：各领域复盘分析
```

**大A**
- 分析今日A股实际表现 vs 预测差异
- 总结成功/失败原因

**老美**
- 分析美股隔夜表现对今日的影响
- 评估中概股/港股联动

**军机处**
- 分析宏观因素对预测的影响
- 评估地缘政治/政策影响

### Phase 2: 因子复盘（2路并发）
```
Agent：量化1哥 + 数据工程师
输入：今日因子表现 + 实际数据
输出：因子有效性分析
```

**量化1哥**
- 计算今日因子IC值
- 分析各因子有效性
- 提出权重调整建议

**数据工程师**
- 验证实际数据准确性
- 检查数据异常
- 输出：数据质量报告

### Phase 3: 误差分析（1路）
```
Agent：因子研究员
输入：预测误差 + 因子表现
输出：误差分析报告
```

- 分析预测误差根本原因
- 识别系统性偏差
- 提出校准系数调整

### Phase 4: 风控复盘（1路）
```
Agent：包青天
输入：今日风控执行 + 实际表现
输出：风控复盘报告
```

- 审查今日风控执行情况
- 分析尾部风险事件
- 评估止损有效性

### Phase 5: 复盘报告 + 明日预测
```
Agent：指挥官
输入：所有Phase输出
输出：复盘报告 + 明日预测
```

**复盘报告**
- 准确率统计（方向/幅度）
- 做对的：总结成功经验
- 需要改进：分析失败原因
- 下一步行动：提出改进措施

**明日预测**
- 基于今日复盘结果校准
- 更新因子权重
- 更新校准系数
- 输出明日Top 6 ETF预测
