# 九章量化局2.0 - 量化分析Skill

## 触发词
- `/quant` + 分析需求
- `分析 [股票代码/名称]`
- `技术面判断 [股票代码]`
- `估值分析 [股票代码]`

## 依赖
- `tdx-connector` MCP（行情数据）
- LLM（MiMo/DeepSeek/OpenRouter）

## 执行流程（DAG并行 + 多空辩论）

### Layer 0：数据获取
```
tdx-connector 拿数据：
  - K线数据（30日）
  - 财务数据（PE/PB/营收/利润）
  - 资金流向（北向资金/主力资金）
```

### Layer 1：并行分析（3个角色同时调用）
```
并行调用3个LLM：
  - 技术分析师：解读MACD/RSI/布林带/支撑阻力
  - 基本面分析师：估值水平/PE百分位/合理价位
  - 资金流向师：主力动向/北向资金/成交量信号
```

### Layer 2：多空辩论（TradingAgents借鉴）
```
串行调用3个LLM：
  - 看涨研究员：挖掘上涨逻辑/利好催化剂
  - 看跌研究员：挖掘下跌风险/利空因素
  - 研究经理：综合多空观点，输出共识结论
```

### Layer 3：风险评估（Qlib借鉴）
```
并行调用3个LLM：
  - 激进风控：关注收益最大化，建议高仓位
  - 中立风控：平衡收益与风险，建议中仓位
  - 保守风控：关注本金安全，建议低仓位
```

### Layer 4：决策输出
```
调用1个LLM：
  - 策略指挥官：综合所有分析，给出最终建议
  - 输出格式：
    ✅ 核心结论（5档评级）
    ✅ 数据支撑（具体数值+来源）
    ✅ 置信度（0-100%）
    ✅ 风险提示（3条）
    ✅ 操作建议（入场/止损/止盈）
```

## 📡 大盘预判增强 (V2.1 新增 — 2026-06-12 实战复盘驱动)

> **背景**：0612实战预判评分35/100，方向全错。复盘五根因驱动以下升级。

### L0：盘前信号层（强制，大盘预判第一步）

**在出任何预判之前，必须先执行L0四维检查**：

| 维度 | 数据源 | 检查项 |
|------|--------|--------|
| 隔夜外盘 | tdx-connector + WebSearch | 美股三大指数涨跌、SOX半导体指数、VIX |
| 亚太早盘 | tdx-connector | 日经/恒生/韩国KOSPI开盘 |
| 重大消息 | wenda_news_query | 政策/地缘/行业/公司 四类 |
| 期货/汇率 | tdx-connector | A50期货、人民币汇率、原油、黄金 |

**铁律**：无L0信号 → **禁止出预判**

### L0.5：催化剂累计评分

对盘前L0发现的催化剂逐一打分：

| 催化剂类型 | 分值 | 示例 |
|-----------|------|------|
| 国家级政策（国务院/发改委） | +8 | 城市更新规划、货币政策转向 |
| 部委级政策（工信部/科技部） | +5 | AI+信息通信意见 |
| 地缘重大变化 | +7 | 美伊停火、贸易协议 |
| 美股科技权重异动(>3%) | +6 | SOX单日涨>5% |
| 行业供需突变 | +4 | 存储涨价、电子布提价 |
| 资金面信号（降准/降息） | +7 | MLF利率调整 |
| 监管层发声 | +3 | 金管总局/证监会 |

**总分判定**：
- ≤10：偏空无阻力，可正常出预判
- 11-25：中性，需综合其他信号
- **>25：禁止判空**（自动将偏空预判改为中性或偏多）
- >35：强制偏多

### L2.5：反身性检查（多空辩论增强）

在Layer 2多空辩论之后，增加反身性检查：

| 条件 | 调整 |
|------|------|
| 大盘连跌≥3天 | 反弹概率 **+30%**，空头论点降权 |
| 大盘连涨≥3天 | 回调概率 **+20%**，多头论点降权 |
| 一致性看空（>70%参与者） | 反转概率 **+25%** |
| 一致性看多（>70%参与者） | 反转概率 **+20%** |
| 周五收盘 | 周末效应，尾盘波动±增加 |

### Beta弹性分层（替代统一±区间）

**不同指数/板块使用不同弹性系数**：

| 分类 | Beta | 日波动区间 | 代表 |
|------|------|-----------|------|
| 高Beta | 1.5-2.5 | ±3-5% | 创业板、半导体、新能源 |
| 中Beta | 1.0-1.5 | ±1.5-3% | 深成指、消费电子、白酒 |
| 低Beta | 0.5-1.0 | ±0.5-1.5% | 上证、银行、公用、高股息 |

### 概率分布三场景（替代单方向预判）

**不再给出单一方向，改为三场景概率分布**：

```markdown
| 场景 | 概率 | 指数区间 | 触发条件 |
|------|------|---------|----------|
| 🟢 乐观 | XX% | XXX-XXX | 若XX条件满足 |
| 🟡 基准 | XX% | XXX-XXX | 基准情景 |
| 🔴 悲观 | XX% | XXX-XXX | 若XX条件满足 |
```

### 量能信号修正

| 量价组合 | 修正信号 |
|---------|---------|
| 连跌+缩量 | **卖盘枯竭**，非买盘乏力 → 反弹信号偏多 |
| 连跌+放量 | **恐慌出逃** → 继续看空 |
| 连涨+缩量 | **追涨意愿不足** → 回调风险 |
| 连涨+放量 | **增量资金进场** → 继续看多 |

### 三大铁律（V2.1硬约束）

1. **无盘前信号不出预判** → L0四维必须完整
2. **催化剂>25分禁止判空** → 自动转中性或偏多
3. **连跌≥3天+一致性看空** → 反转概率+30%，空头降权

### 连续验证看板

每次预判后，必须做验证：

| 日期 | 预判方向 | 实际方向 | 方向分 | 区间分 | 总评 |
|------|---------|---------|--------|--------|------|
| 0612 | 偏空 | 大涨 | 0/30 | 20/70 | 35/100 |
| 下次 | ... | ... | ... | ... | ... |

**目标**：5次预判内将综合评分从35提升至60+。

### L0.8：风格轮动信号（在L0盘后、L0.5催化剂之后执行）

地缘/宏观事件对风格的差异化影响：

| 事件类型 | 进攻型资产 | 防御型资产 | 逻辑 |
|---------|-----------|-----------|------|
| 地缘缓和/和平 | 🟢 大利好 | 🔴 利空出尽 | 避险资金撤出防御→涌入进攻 |
| 地缘升级/战争 | 🔴 利空 | 🟢 大利好 | 避险资金涌入防御 |
| 降息/宽松 | 🟢 利好 | ➡️ 中性 | 成长估值弹性大 |
| 加息/收紧 | 🔴 利空 | 🟢 利好 | 价值股抗跌 |
| 科技突破 | 🟢 大利好 | ➡️ 中性 | 产业逻辑驱动 |
| 衰退担忧 | 🔴 利空 | 🟢 利好 | 防御优先 |
| 政策刺激 | 🟢 利好 | ➡️ 中性偏多 | 周期/成长更受益 |

### 风格分类标签（每个观察池标的必须标注）

| 标签 | Beta | 代表标的 |
|------|------|---------|
| 🚀 进攻-科技 | 高β(1.5-2.5) | 立讯精密/中芯国际/北方华创/紫光国微/半导体ETF |
| ⚡ 进攻-新能源 | 高β(1.5-2.0) | 宁德时代/隆基绿能/光伏ETF |
| 🛡️ 防御-金融 | 低β(0.5-1.0) | 工商银行/兴业银行/中国平安 |
| 🏛️ 防御-公用 | 低β(0.3-0.8) | 长江电力 |
| 🍷 消费 | 中β(0.8-1.2) | 贵州茅台 |
| 🌾 防御-农业 | 低β(0.5-0.8) | 农业ETF |
| ✈️ 进攻-军工 | 中高β(1.0-1.5) | 军工ETF |

### 避险→进攻转换检查（V2.2铁律4）

| 条件 | 信号 |
|------|------|
| 地缘缓和 + 科技映射利好 | 防御型资产预判**降级**，从"偏多"改为"中性"或"偏空" |

## 🌐 美股联动因子量化 (V2.3 新增 — 2026-06-15 升级驱动)

> **背景**：0612复盘根因#1就是隔夜SOX+7.9%被完全遗漏。V2.1虽增加L0盘前信号层，但只是"有/无"检查，未量化映射系数。0615隔夜美股涨→A股半导体+5.9%，映射关系明确但V2.2仍为定性判断。

### L0.6：美股→A股联动因子（在L0盘前检查中增加量化环节）

**传导路径与权重系数**：

| 美股因子 | A股映射目标 | 映射权重 | 传导时滞 | 说明 |
|---------|------------|---------|---------|------|
| 费城SOX半导体指数 | 半导体ETF/北方华创/中芯国际 | **0.7** | 隔夜0.8x / 早盘0.5x / 午后0.3x | 最强映射链 |
| 纳斯达克综合指数 | 创业板指/成长ETF/消费电子 | **0.6** | 隔夜0.8x / 早盘0.5x / 午后0.3x | 次强映射链 |
| 标普500指数 | 沪深300/上证50/大盘蓝筹 | **0.4** | 隔夜0.7x / 早盘0.4x / 午后0.2x | 基准映射链 |
| VIX恐慌指数 | 全市场反向指标 | **反向-0.5** | VIX>25看空 / VIX<15看多 | 波动率映射 |
| 美元指数(DXY) | 人民币汇率→北向资金 | **-0.3** | DXY↑→RMB↓→北向流出 | 间接映射链 |

**联动计算公式**：

```
联动贡献分 = Σ(美股因子涨跌幅 × 映射权重 × 时滞系数)

时滞系数：
  隔夜期间(15:30-21:30北京时间)：0.8（消化时间充足，充分反映）
  早盘30分钟(9:30-10:00)：0.5（开盘集中反映）
  午盘(10:00-15:00)：0.3（已被消化，边际递减）
```

**联动判定规则**：

| 联动贡献分 | 判定 | 对预判的影响 |
|-----------|------|------------|
| >+2.0% | 🟢 强正面联动 | 牛市场景概率+15%，区间上限上移 |
| +0.5%~+2.0% | 🟡 弱正面联动 | 牛市场景概率+5%，中性偏多 |
| -0.5%~+0.5% | ⚪ 中性联动 | 不调整 |
| -2.0%~-0.5% | 🟠 弱负面联动 | 熊市场景概率+5%，中性偏空 |
| <-2.0% | 🔴 强负面联动 | 熊市场景概率+15%，区间下限下移 |

**特殊规则**：
- SOX单日波动>5% → 强制调整半导体ETF预判区间（±2%额外弹性）
- 纳斯达克单日波动>3% → 强制调整创业板预判区间（±1.5%额外弹性）
- 美股+A股同时大跌 → 恐慌共振，反弹概率+20%（反身性增强）

> **背景**：V2.1/V2.2的技术分析仍为"描述性"（如"MACD金叉""RSI超买"），缺乏统一量化打分。导致不同标的之间技术面无法横向比较，也无法与其他模块（催化剂、风格轮动）做加权合成。

### L1.2：技术指标量化打分（在七层漏斗之前执行）

**六大指标系统化打分（总分100分）**：

| # | 指标 | 满分 | 看多信号(+满分) | 看空信号(0分) | 中性(+半分) | 冲突(-1/4分) |
|---|------|------|--------------|-------------|-----------|------------|
| 1 | MACD | 20 | DIF上穿DEA(金叉) + MACD柱由负转正 + DIF>0 | DIF下穿DEA(死叉) + MACD柱由正转负 + DIF<0 | DIF接近DEA但未交叉 | 金叉但柱缩短 / 死叉但柱缩短 |
| 2 | RSI(14) | 15 | RSI 40-60且拐头向上 / RSI从<30超卖区回升 | RSI>70超买 / RSI从高位拐头向下 | RSI 45-55区间震荡 | 多周期RSI方向不一致 |
| 3 | KDJ | 15 | K上穿D(金叉) + J>0 + K/D<80低位金叉 | K下穿D(死叉) + J>100钝化后下 | K/D 40-60区间交叉 | KDJ与价格背离 |
| 4 | 布林带(20,2) | 15 | 价格触及下轨后反弹 / 带口缩窄后扩张向上 | 价格触及上轨后回落 / 带口缩窄后扩张向下 | 价格在中轨附近 | 突破上/下轨后持续 |
| 5 | 均线系统 | 20 | MA5>MA10>MA20多头排列 / 价格站上所有均线 | MA5<MA10<MA20空头排列 / 价格跌破所有均线 | 均线缠绕交叉 | 价格与均线方向背离 |
| 6 | 成交量 | 15 | 放量上涨(量增价升) / 缩量回调(量缩价跌-健康) | 放量下跌(量增价跌) / 缩量反弹(量缩价涨-无力度) | 量能平稳 | 量价关系混乱 |

**总分判定**：

| 技术总分 | 信号 | 对预判的影响 |
|---------|------|------------|
| >70 | 🟢 技术面看多 | 三场景中牛市场景概率+10%，区间上限上移 |
| 55-70 | 🟡 偏多 | 中性偏多 |
| 40-55 | ⚪ 中性 | 不调整 |
| 25-40 | 🟠 偏空 | 中性偏空 |
| <25 | 🔴 技术面看空 | 三场景中熊市场景概率+10%，区间下限下移 |

**多周期一致性加分/减分**：

| 条件 | 调整 |
|------|------|
| 日/周/月线三周期同方向 | 额外±5分 |
| 日周同向月线反向 | 额外±2分 |
| 三周期全部方向不同 | 额外-3分（混乱信号） |

### 技术指标×风格轮动交叉验证

| 组合 | 判定 | 逻辑 |
|------|------|------|
| 技术看多 + 风格偏进攻 | 🟢 强看多(叠加) | 技术面与风格共振 |
| 技术看多 + 风格偏防御 | ⚠️ 弱看多 | 防御风格削弱科技技术面 |
| 技术看空 + 风格偏进攻 | ⚠️ 弱看空 | 风格偏多可能抵消技术面 |
| 技术看空 + 风格偏防御 | 🔴 强看空(叠加) | 技术面与风格共振向下 |

## 五大铁律（V2.3完整版）

1. **无盘前信号不出预判** → L0四维必须完整
2. **催化剂>25分禁止判空** → 自动转中性或偏多
3. **连跌≥3天+一致性看空** → 反转概率+30%，空头降权
4. **风格轮动必须判断** → 地缘缓和时防御降级，地缘升级时进攻降级
5. **美股联动>2%必须量化** → SOX/Nasdaq异动必须计算联动贡献分并调整预判区间
