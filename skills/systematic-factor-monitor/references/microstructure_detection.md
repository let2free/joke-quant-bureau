## 检测步骤详解

### Step 2.1: 指数事件日历检测

**目标**：识别未来30天内的指数审核生效事件。

**执行步骤**：

```
1. 搜索关键词（使用WebSearch）：
   - "中证指数样本调整 2026年6月"
   - "科创50样本调整 2026年6月"
   - "创业板指样本调整 2026年6月"
   - "MSCI中国指数调整 2026年5月"
   数据源：中证指数官网（www.csindex.com.cn）、上交所/深交所官网、Wind/同花顺新闻

2. 若发现未来30天内有指数审核生效事件：
   a. 记录以下信息：
      - 指数名称
      - 生效日期
      - 新增成分股列表（股票代码+名称）
      - 删除成分股列表
      - 权重变化（新增股权重、删除股权重）

   b. 识别受益/受损ETF：
      - 科创50调整 → 科创50ETF(588000) 受益
      - 创业板调整 → 创业板ETF(159915) 受益
      - 中证500调整 → 中证500ETF(510500) 受益
      - 沪深300调整 → 沪深300ETF(510300) 受益

   c. 计算被动资金冲击系数：
      公式：
      冲击系数 = |新权重 - 旧权重| × 跟踪该指数的基金规模 / 该股近20日日均成交额
      
      阈值判定：
      - 冲击系数 > 3 → 🔴 强烈买入/卖出信号
      - 冲击系数 1-3 → 🟡 中度信号
      - 冲击系数 < 1 → 🟢 弱信号
      
      示例：
      某股票被纳入科创50，新权重2%，旧权重0%
      跟踪资金规模 = 1000亿元
      近20日日均成交额 = 5亿元
      → 冲击系数 = 2% × 1000亿 / 5亿 = 4.0（强烈买入信号）

3. 输出"调仓窗口"标记：
   - 生效日前5个交易日 → 启动"调仓窗口"标记
   - 在ETF推荐中，对应ETF排序权重 +15%
   - 若冲击系数 > 3 → 权重 +25%
```

### Step 2.2: 被动资金流检测（每日）

**目标**：检测指数生效日前后的被动资金流入压力。

**执行步骤**：

```
1. 检查未来5个交易日是否有指数审核生效

2. 若有 → 计算"被动资金流入压力"：
   公式：
   日流入压力 = 跟踪资金规模 × 调仓比例 / 剩余交易日
   
   示例：
   科创50 6月14日生效
   跟踪资金 = 1000亿元
   调仓比例 = 新增成分股合计权重 = 8%
   剩余交易日 = 3天（假设今天6/11）
   → 日流入压力 = 1000亿 × 8% / 3 = 26.7亿元/日

3. 判定：
   - 日流入压力 > 1亿元 → 强烈买入信号（对成分股）
   - 日流入压力 0.5-1亿元 → 中度买入信号
   - 日流入压力 < 0.5亿元 → 弱信号

4. 检查"抢跑"行为：
   - 搜索"龙虎榜 [成分股名称] 2026年X月"
   - 搜索"大宗交易 [成分股名称] 2026年X月"
   - 若连续3日出现同一成分股的大宗交易 → "抢跑"信号（主动基金提前布局）
```

## 注意事项

1. **指数审核公告可能提前1-2个月发布** —— 在公告发布后，主动基金可能"抢跑"，因此检测窗口应提前至公告日（而非生效日）
2. **冲击系数计算公式假设"资金均匀流入"** —— 实际中，80%资金在生效日前1-2个交易日集中流入，波动可能更大
3. **若多个指数同时审核** —— 需计算"复合冲击系数"，即将各指数冲击系数求和
4. **ETF本身也可能被调整** —— 若ETF被纳入某个指数（如沪深300纳入科创50ETF），则产生"ETF层面的被动资金流入"
