## 检测步骤详解

### Step 1.1: 央行OMO/MLF操作检测

**目标**：判断央行近期操作是否"超预期"

**执行步骤**：

```
1. 搜索近5个交易日央行OMO/MLF操作公告
   搜索关键词：
   - "人民银行 逆回购 2026年X月X日"
   - "MLF操作 2026年X月"

2. 提取以下字段：
   - 操作量（亿元）
   - 操作利率（%）
   - 期限（7天/14天/1年期等）
   - 是否有"超预期"市场评论（搜索"逆回购 超预期"）

3. 计算"超预期程度"：
   方法A（有Wind/同花顺预期数据）：
     超预期程度 = (实际操作量 - 市场预期中值) / 市场预期中值
   
   方法B（无预期数据，用历史均值）：
     超预期程度 = (实际操作量 - 前3月均值) / 前3月均值

4. 判定：
   - 超预期程度 > +20% → 🟢 强烈宽松信号
   - 超预期程度 0~+20% → 🟢 宽松信号
   - 超预期程度 -20%~0 → 🟡 中性
   - 超预期程度 < -20% → 🔴 收紧信号
```

**注意事项**：
- 若5日内无OMO/MLF操作 → 不代表中性，需结合DR007判断
- 若到期量 > 操作量（净回笼）→ 实际是收紧，需计算"净投放 = 操作量 - 到期量"

---

### Step 1.2: DR007利率检测

**目标**：通过银行间利率判断市场实际流动性

**执行步骤**：

```
1. 搜索"DR007 收盘 2026年X月X日"
   数据源：中国货币网（chinamoney.com.cn）

2. 提取：
   - 当日DR007加权均价（%）
   - 近5日DR007走势（上行/下行/平稳）

3. 对比政策利率：
   政策利率 = 7天逆回购利率（当前1.50%，2026年6月）
   
   判定：
   - DR007 < 政策利率 - 10bp → 🟢 流动性宽松
   - DR007 在政策利率±10bp区间内 → 🟡 中性
   - DR007 > 政策利率 + 20bp → 🔴 流动性收紧（市场资金紧张）

4. 趋势判定：
   - 近5日DR007持续下行 → 🟢 宽松趋势
   - 近5日DR007持续上行 → 🔴 收紧趋势
   - 震荡 → 🟡 中性
```

**政策利率参考数据**：

| 时期 | 7天逆回购利率 | 备注 |
|------|----------------|------|
| 2025年Q4 | 1.70% | - |
| 2026年Q1 | 1.50% | 2025年12月降息20bp |
| 2026年Q2 | 1.50% | 维持不变 |

### Step 1.3: 北向资金流向检测

**目标**：判断外资对A股的态度变化

**执行步骤**：

```
1. 使用TDX工具查询北向资金（代码：869998）
   或搜索"北向资金 净流入 2026年X月X日"

2. 提取近5日每日净流入金额（亿元）
   
3. 计算：
   a. 连续净流入天数
   b. 累计净流入金额
   c. 冲击系数 = 累计净流入 / A股自由流通市值
      A股自由流通市值 ≈ 45万亿元（2026年6月估算）
      冲击系数 > 0.5% → 强力看多
      冲击系数 0.2-0.5% → 中度看多
      冲击系数 < 0.2% → 弱信号

4. 判定：
   - 连续5日净流入 AND 冲击系数 > 0.2% → 🟢 外资回流，强烈看多
   - 连续3-4日净流入 → 🟢 偏多
   - 净流入/出反复 → 🟡 中性
   - 连续3日以上净流出 → 🔴 外资撤离，偏空
```

**注意事项**：
- 北向资金数据自2024年8月起改为"季度公布"，实时数据可能不准确
- 若无法获取准确北向数据 → 用"沪深港通资金流向"新闻替代，或标注"数据不可用"

### Step 1.4: 综合流动性状态判定

**目标**：将多个信号合成一个"流动性状态"

**判定逻辑**：

```
IF (OMO超预期程度 > +20%) OR (DR007 < 政策利率-10bp) OR (北向连续5日净流入 AND 冲击系数>0.2%)
  → 流动性状态 = 🟢 宽松

ELSE IF (OMO超预期程度 < -20%) AND (DR007 > 政策利率+20bp)
  → 流动性状态 = 🔴 收紧

ELSE
  → 流动性状态 = 🟡 中性
```

**输出格式**：

```yaml
liquidity_status: "🟢宽松"
confidence: 85%  # 三个信号一致则高置信度，分歧则低
signals:
  - name: "OMO操作"
    value: "超预期+35%"
    signal: "🟢宽松"
  - name: "DR007"
    value: "1.35%（政策利率-15bp）"
    signal: "🟢宽松"
  - name: "北向资金"
    value: "连续5日净流入152亿，冲击系数0.34%"
    signal: "🟢宽松"
```
